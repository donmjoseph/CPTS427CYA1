# CYA 01 – Jamf MDM Implementation @ WSU IT

## Chosen Adventure

**Stretch Goal at Part-Time Employment**

As a Technical Assistant at Washington State University IT, I was tasked with
implementing and configuring Jamf Pro MDM (Mobile Device Management) to manage
Apple devices across the department — bringing them into compliance with
Washington State law and aligning their management posture with existing
Windows infrastructure.

## Summary of Deliverable

Over approximately 15–20 hours of work, I accomplished the following:

- **Built Apple enrollment profiles** in Jamf Pro to enable zero-touch / automated
  device enrollment
- **Designed and deployed configuration profiles** governing security policies
  (screen lock, FileVault encryption, password complexity, network restrictions)
  in alignment with Washington State IT security requirements
- **Scoped and assigned** profiles to device groups reflecting the department's
  organizational structure
- **Documented the full configuration process** in the WSU internal Confluence
  knowledge base for use by other IT staff
- **Enrolled physical devices** — currently deployed and actively managed devices
  are visible in Cortech (the Apple reseller/service center in the Compton Union
  Building at WSU)
- **Ongoing:** Automating Splunk and Cortex XDR deployment to managed Apple
  devices via Jamf policy scripting, to match the automated deployment pipeline
  already in place for Windows devices
